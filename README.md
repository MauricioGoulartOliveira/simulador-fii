# 📊 Simulador de FIIs (Fundos Imobiliários) - Excel

Este projeto é um **simulador de investimentos em FIIs (Fundos de Investimento Imobiliário)** desenvolvido em **Microsoft Excel**. Ele permite simular e analisar diferentes cenários de aportes mensais, rendimentos e crescimento de patrimônio ao longo do tempo.

## 🧾 Funcionalidades

- 💰 Cálculo de rendimentos mensais com base nos aportes realizados.
- 📈 Simulação de crescimento do patrimônio com reinvestimento dos rendimentos.
- 📊 Gráficos automáticos de evolução do patrimônio e dos proventos.
- 🖌️ Planilha estilizada para melhor visualização e usabilidade.

---

## 📌 Como usar

1. **Baixe o arquivo** `simulador-fii.xlsx`.
2. **Abra no Microsoft Excel** (ou outro software compatível com `.xlsx`).
3. **Preencha os campos** na aba de entrada:
   - Valor inicial do investimento
   - Aporte mensal
   - Rendimento mensal (%)
   - Período da simulação (meses)
4. Visualize automaticamente os resultados e gráficos gerados.

---

## 📁 Estrutura da planilha

- **Simulação**: onde o usuário insere os dados da simulação.
- **Resultados**: mostra a evolução do patrimônio e dos rendimentos.
- **Gráficos**: visualização dos resultados de forma dinâmica.
- **Tabelas auxiliares**: utilizadas para cálculos intermediários.

---

## 💡 Exemplos de uso

Você pode usar este simulador para:

- Planejar sua aposentadoria com FIIs
- Comparar cenários de aportes diferentes
- Ver o impacto do reinvestimento dos rendimentos
- Simular a renda passiva futura com base no seu perfil de investidor
